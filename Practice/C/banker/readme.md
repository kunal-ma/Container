Altis Banking Systems
======

TUI-based multipurpose banking and management software for financial institutions, written in C. Built using C17 and only Standard C libraries, it is one of the most efficient, optimized and intuitive programs currently available in the market.

---

## Features

Altis Banking Systems is heavily optimized to fulfill the current industry requirements :

* **User Account Management:** Allows adding, modifying and removing user accounts, and stores all data in an external file to avoid manual input of records.

* **Transactions:** Support for conversion of digital currency to cash, i.e. via deposit and withdrawal of money from user accounts, with Token ID based transaction logging.

* **Loan Management:** Allows sanctioning and tracking of loans and fixed deposits, including systematic background checks and eligibility.

* **Stock Market:** [Upcoming] Support for Stock trading and market tracking, including management and handling of Dematerialized accounts, and customized daily insight for users.
